# eurocup
Euro cup champion predict and game result present

Based on d3.js and jquery.js.
Data can be refreshed every hours base on gamble website.

2016 欧洲杯 冠军预测以及比赛结果展示。

数据来源于赌博网站，可以每隔一小时更新一次。
